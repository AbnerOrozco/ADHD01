ejemplos SASS

utizando (@if, @else if, @else) y es una custom box
para lo grar ese estilo podemos usar los siguientes estilos
$theme: 'dark'; // Prueba con 'light', 'dark', 'colorful', etc. cambiando solo una palabra


para usar esto utilizacmos lo siguiente

@mixin box-theme($theme)
{ @if $theme == 'light' {
background-color: #f8f9fa;
color: #000;
&:hover {
background-color: #e2e6ea;

} } @else if $theme == 'dark' {
background-color: #343a40;
color: #fff;

&:hover {
background-color: #23272b;
} } @else if $theme == 'colorful' {
background-color: #ff6347;
color: #fff;

&:hover {
background-color: #e5533a;
}
} @else {
background-color: #dcdcdc;
color: #000;

&:hover {
background-color: #c0c0c0;
} }
